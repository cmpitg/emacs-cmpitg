== How to run

* Environment variables:
** `EMACS_PORT`
** `EMACS_FORCE_TOGGLE_DEBUG_ON_ERROR`
** `EMACS_NO_EXPERIMENTAL`
** `EMACS_ENABLED_MAIL`
** `EMACS_ENABLED_NOTES`

* High-level entry point: `rmacs`

* Low-level entry point: `emacs-cmpitg`

=== Emacs as text editor

[source,sh]
----
rmacs edit <path>
----

=== Emacs as mail browser (mu4e)

[source,sh]
----
rmacs mail
----

=== Emacs as note taker

[source,sh]
----
rmacs notes
----
